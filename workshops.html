workshops

On 20th of April, just before the main conference day, we'll host multiple workshops targeted both at the React newcomers and hardcore fans. Take the full advantage of your conference visit and get training from our best speakers and trainers.

Trainings will be held at IIRE Conference Center, Lombokstraat 40, 1094 AL Amsterdam.
Starting from 9:00 in the morning, till 17:00.

  • Max Stoiber
     

    React Workshop with Max Stoiber

    This workshop will get you up to speed with React just in time for React Amsterdam Conference. Learn everything you need to know, including ES6, Component Thinking, JSX, State Management and much more. Dive straight in with hands-on labs that'll help you grasp the concepts in less time than it takes to drink a coffee!

    Table of contents

    - Setup and ES6 basics
    - React intro (ReactDOM/React, Components, JSX,...)
    - Interaction and State Management
    - Lifecycles

    Prerequisites

    Bring a laptop (preferably *nix based, but we can work with Windows if necessary) with Node.js, npm and git pre-installed.
    Basic HTML, CSS and JavaScript knowledge is required.

    Workhop level

    Beginner (React) -> Intermidiate
